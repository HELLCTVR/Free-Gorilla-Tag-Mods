# Free-Gorilla-Tag-Mods (Description)
Get Free Gorilla Tag Mod Menus, Trying To Always Keep To Date, And Also Announcing On A Discord Server The Status Of The Mods (New Update Or Forgoten).

Every Mods Should Be At The Latest Update And All Ready To Be Used!

# How To Install ;
Step By Step ;
1- Go To Any Browser And Search "Monke Mod Manager", Or Just Copy And Paste This Link ; https://github.com/arielthemonke/MonkeModManager 
2- Open The "Monke Mod Manager" File And, It Should Automaticly Put It To Gorilla Tag, If You Have It, If It Doesn't, Just Press The Little Three Dots And Browse Trough These File ; C:\Program Files (x86)\Steam\steamapps\common\Gorilla Tag\
3- Press The First Two Check Box (Utila and BepInEx)
# That's It !

# How To Install The Mods ;
Step By Step ;
1- Go In The "plugins" Folder (Gorilla Tag --> BepInEx --> plugins)
2- You Can Create Folder To Organise The Stuff, But Just Put The ".dll" File In There.
# That's It !
